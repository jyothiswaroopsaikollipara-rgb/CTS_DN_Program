# Singleton Pattern

## Objective
Implement Singleton Design Pattern to ensure only one object is created.

## Concepts Used
- Private Constructor
- Static Variable
- Static Method
- Lazy Initialization

## Files
- Singleton.java
- Main.java

## Output

Both objects have same hashcode, proving only one object is created.
