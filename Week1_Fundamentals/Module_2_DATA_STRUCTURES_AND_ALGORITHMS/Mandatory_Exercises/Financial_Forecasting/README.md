# Financial Forecasting

## Objective

Predict future value of an investment using recursion.

## Concepts Used

- Recursion
- Base Case
- Recursive Case
- Time Complexity

## Files

- ForecastCalculator.java
- ForecastMain.java

## Example

Initial Amount = ₹1000

Growth Rate = 10%

Years = 5

Future Value = ₹1610.51

## Complexity

Time Complexity : O(n)

Space Complexity : O(n)

## Applications

- Investment Forecasting
- Revenue Prediction
- Population Growth
- Sales Forecasting
