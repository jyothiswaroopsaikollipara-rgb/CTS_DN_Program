# Module 5 - Spring Core and Maven

## Overview

This module introduces the fundamentals of the Spring Framework, including Spring Core, Dependency Injection (DI), Inversion of Control (IoC), and Maven project configuration.

---

## Mandatory Exercises

### Exercise 1: Configuring a Basic Spring Application

**Concepts Used**

* Spring Core
* IoC Container
* Bean Configuration
* Java-based Configuration (`@Configuration`, `@Bean`)

**Files**

* GreetingService.java
* GreetingServiceImpl.java
* SpringConfig.java
* Main.java

---

### Exercise 2: Implementing Dependency Injection

**Concepts Used**

* Dependency Injection (Constructor Injection)
* Bean Wiring
* Spring IoC Container

**Files**

* Engine.java
* Car.java
* SpringConfig.java
* Main.java

---

### Exercise 3: Creating and Configuring a Maven Project

**Concepts Used**

* Maven
* pom.xml
* Spring Dependencies

---

## Technologies Used

* Java 25
