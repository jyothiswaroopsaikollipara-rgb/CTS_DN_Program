# Week 2 – Spring Framework

## Overview

This week focused on learning the fundamentals of the Spring Framework and building database-driven applications using Spring Boot, Spring Data JPA, and Hibernate. The exercises covered Dependency Injection, Maven project configuration, Object Relational Mapping (ORM), and CRUD operations using an H2 database.

---

## Modules Completed

### Module 5 – Spring Core and Maven

#### Objective

Understand the core concepts of the Spring Framework and build Maven-based Spring applications.

#### Topics Covered

* Introduction to Spring Framework
* Spring Core
* Inversion of Control (IoC)
* Dependency Injection (DI)
* Spring Beans
* Bean Configuration
* Maven Project Setup

#### Outcomes

* Created a Maven-based Spring project.
* Configured Spring beans.
* Implemented Dependency Injection.
* Understood IoC Container.
* Successfully built and executed the application.

---

### Module 6 – Spring Data JPA with Spring Boot & Hibernate

#### Objective

Develop a Spring Boot application using Spring Data JPA and Hibernate to perform CRUD operations on an H2 database.

#### Topics Covered

* Spring Boot
* Spring Data JPA
* Hibernate ORM
* Entity Mapping
* Repository Layer
* Service Layer
* H2 Database
* CRUD Operations

#### Outcomes

* Created an Employee entity.
* Implemented Repository and Service layers.
* Connected the application to an H2 database.
* Performed Create, Read, Update, and Delete operations using `JpaRepository`.
* Verified SQL generation through Hibernate.
* Tested the application using the H2 Console.
