# Week 3 - Module 2 - REST Using Spring Boot

## Objective

Develop RESTful web services using Spring Boot for Employee and Department data.

---

## Technologies Used

- Java 25
- Spring Boot 3.5.x
- Spring Web
- Spring Core (XML Configuration)
- Maven

---

## Project Structure

- Bean
- DAO
- Service
- Controller
- XML Configuration

---

## REST APIs

### GET /employees

Returns the list of employees loaded from `employee.xml`.

### GET /departments

Returns the list of departments loaded from `employee.xml`.
