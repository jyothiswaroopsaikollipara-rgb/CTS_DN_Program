# Week 5 – React

This folder contains the mandatory hands-on exercises completed as part of Week 5 of the Cognizant Digital Nurture 5.0 Java FSE Deep Skilling Program.

## Mandatory Hands-on Exercises

### 1. myfirstreact
- Created a React application using Create React App.
- Displayed the welcome message for the first React session.

### 2. studentapp
- Created a Student Management Portal.
- Implemented Home, About, and Contact components.
- Rendered multiple React components.

### 3. scorecalculatorapp
- Created the `CalculateScore` functional component.
- Passed student details using props.
- Calculated and displayed the average score.

### 4. blogapp
- Implemented React component lifecycle methods.
- Used `componentDidMount()` to fetch blog posts.
- Displayed post titles and content.
- Implemented `componentDidCatch()` for error handling.

### 5. cohorttracker
- Styled React components using CSS Modules.
- Applied the required box styling to cohort details.
- Used inline styling for conditional heading colors.
- Displayed ongoing cohorts in green and other cohorts in blue.
